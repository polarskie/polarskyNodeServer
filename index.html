pol-webGL simplified API

This is a simple API for webGL. You can use it for web 3D game development for free, but you will have to notify the origination is from POLARSKY's work. Press here to have a quick view about this API. This API library is still not robust, and you may find a lot of error and implementation that is not formal in it. If you want to contribute to it, fork this library on github and implement your correction or optimization. Your support is very invaluable!

2D code(QR code) magic

This is a cooperation project. We tried to combine a picture with a 2D code so that a 2D code picture provide much more information than it was designed to. The basic idea is to mimic the original 2D code by always fill the center pixels of every black or white blocks in color of the original code. Thus the scanner will simply determine the 2D code by the center pixels. There are also some other tricks used here, for example: sacrificing some blocks is OK. I must note here that this is not a orginal idea. If you are interested in it, you can find much more information here. What we have done is implement it and find some ways to enhance the performance.

A WECHAT public account

When you fetch a list of resources, the response includes a subset of the attributes for that resource. This is the “summary” representation of the resource. (Some attributes are computationally expensive for the API to provide. For performance reasons, the summary representation excludes those attributes. To obtain those attributes, fetch the “detailed” representation.)
